Suite

Workflow

Desde detectar un problema hasta dejar una solucion documentada.

La suite tiene sentido porque tambien ordena el recorrido operativo completo.

01

Descubrir

El Hub CLI centraliza inventario, ejecucion y chequeos de salud para que cada app sea facil de ubicar y operar.

02

Actuar

Cada microapp ataca un caso real: datos, logs, cloud, Git, YAML, migracion o aprendizaje tecnico.

03

Estandarizar

Docker, manifests, health checks, Makefile y docs-first convierten la suite en algo repetible y evaluable.

04

Escalar

MCP local, GitHub Pages y documentacion viva amplian la suite hacia asistentes IA, demos y presentacion publica.

Inicio rapido

# instalar dependencias PHP
make install

# levantar entorno local
make up

# listar microapps
make hub-list

# ejecutar chequeo general
make hub-doctor

Canales de uso

  • Navegador: dashboard central y microapps web.
  • CLI: Hub para discovery, run y health checks.
  • Contenedores: Docker para portabilidad y demo rapida.
  • IA local: servidor MCP de solo lectura para contexto seguro.
