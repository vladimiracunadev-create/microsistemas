Quick Start y Hub CLI

El producto se ve mejor cuando queda claro como se prueba y como se usa.

Inicio rapido

# instalar dependencias PHP
make install

# levantar la aplicacion con Docker
make up

# servir en local
make serve

Comandos del Hub

make hub-list
make hub-run APP=Conversor
make hub-up APP=CapacitySim
make hub-doctor
make smoke

Microapps destacadas

Una muestra corta y clara para entender el alcance de la suite sin sobrecargar la pagina.

AWS Assistant Pro

Recetas guiadas para AWS CLI con enfoque operativo y lectura educativa.

CapacitySim

Simula capacidad, RPS y costos para planificar infraestructura antes del despliegue.

SqlViewer

Inspeccion de base de datos sin clientes pesados y con acceso rapido desde navegador.

PhpMigrator

Reduce la friccion entre codigo PHP heredado y una base mas moderna.

GitTrainer

Biblioteca visual para aprender y resolver escenarios Git con mas confianza.

KatasMultiLang

Comparador side-by-side para estudiar patrones en varios lenguajes y frameworks.
